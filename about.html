About

This website is made to help you understand how a particular algorithm works.

It provides:

  • Visualizing different sorting algorithms.
  • Controlling the speed of the visualization.
  • User defined inputs.
  • Comparisons and passes count.
  • Displaying a trace code -- which means as the visualization is going on it will heighlight which code is being executed.
  • Time complexity analysis.
  • It is a responsive design -- can be used on mobile, tablet, laptop or desktop PCs.
