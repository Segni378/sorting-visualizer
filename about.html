About

This website is developed by Segni Guta, a 3rd computer engineering student.

It is developed using html, css and vanilla javascript. It is intended to help students understand how a particular algorithm works. Eventhough it is not a new concept, the developer tried to improve the sorting visualizers available on the internet.

This website provides various services like:

  • Visualizing the sorting algorithms.
  • Controlling the speed of the working of the visualization.
  • User defined input of numbers.
  • Counting number of comparisons and passes.
  • Displaying a trace code -- which means as the visualization is going on it will heighlight which code is being executed.
  • Time complexity analysis.
  • It is a responsive design -- can be used on mobile, tablet, laptop or desktop PCs.

The developer thinks to make it more useful to students in the future. Please feel free to use it as long as you want.
